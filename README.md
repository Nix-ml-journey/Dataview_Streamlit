# Dataview_Streamlit

I perform data cleaning and EDA on Kaggle's "The World's Billionaires by Forbes" dataset, then train regression models to generate predictions. The results are presented in an interactive Streamlit app for intuitive analysis and visualization.

I also use Cursor to help me with coding.

## Update for the things that I made

I have created some files that will load the dataset, and I have tested them - everything works. I use YAML for the config because it's easier to manage by adding or removing values. The UI.py file serves as the UI design for Streamlit, and main.py is the main Python code that will contain the prediction model and the function to run the main Streamlit file. The structure will change over time as the project evolves.

Today I focused on setting up the project - most of my time was spent connecting to Git so that I can push changes to it every time I work on this project.
